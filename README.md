## How to Run
1. Clone the repository:
```bash
git clone https://github.com/jrsrezende/agenda-web.git
```
2. Install the dependencies
```bash
npm i
```
3. Compile the Angular application to run locally at http://localhost:4200
```bash
ng s -o
```
